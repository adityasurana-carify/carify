- CTA button: "Book a Demo"

**Hero Section:**
- Headline: "Let us save your time and costs with Carify Agents"

**Interactive ROI Calculator:**
Create an interactive calculator with the following inputs:
1. Average Call Time (in Mins) Per Patient - slider (0-60 mins)
2. Total number of patients annually - number input
3. Total number of calls per patient annually - number input

- Estimated time saved (in hours)
- Total hours saved annually
- Total dollars saved annually (using $20/hour default rate)

Display outputs prominently with large numbers and currency formatting.

Add disclaimer text: "The calculator assumes a default industry average rate of $20 per hour for the call handling role to compute the Total Dollars Saved"
